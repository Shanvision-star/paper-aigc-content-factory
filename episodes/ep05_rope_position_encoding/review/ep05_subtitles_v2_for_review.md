# EP05 RoPE 字幕稿 V2 待审核

## 字幕原则

- 字幕不逐字重复口播，而是做动态字幕：主字幕跟口播主干同步，辅助字幕补机制或画面关键词。
- 这里的“动态”指固定字幕槽按下方时间轴显示、替换或高亮文本；不让字幕块移动，不用进度条替代字幕。
- 每条字幕优先承载一个机制点：加法、旋转、点积、相对距离、公开边界。
- 主字幕建议 14-24 个汉字；辅助字幕建议 10-22 个汉字。若超过两行，拆下一条。
- 专业词第一次出现时保留英文和中文解释；公式符号要和口播、画面一致。

## 字幕审核风险

- 不把“RoPE 主流”写成“所有模型都用 RoPE”。
- 不把“旋转 Q/K”写成“token 在句子平面转圈”。
- 不把 DeepSeek-V4 的 hybrid attention / Partial RoPE 简化成普通 RoPE。
- 不在字幕里出现未审核的专有模型内部实现。
