# EP05 RoPE 字幕稿 V3 216s 待审核

## 字幕原则

- 本版按 216 秒慢讲设计，最终成片不得低于 180 秒。
- “动态字幕”指固定字幕槽按时间显示、替换和轻量高亮；不是进度条，也不是让文字、公式或字幕块移动。
- 主字幕跟随口播主干，辅助字幕只补机制关键词；字幕不遮挡公式、Q/K 圆盘、来源标注和 MATLAB 动画主体。
- 每条主字幕优先 12-22 个汉字；复杂公式段允许辅助字幕出现符号读法。
- 视觉符号和口播表达必须一致：`mθ_i` 口播为“m 乘 theta 下标 i”，`nθ_i` 口播为“n 乘 theta 下标 i”。

## 字幕审核风险

- 不写“点积后只剩 `n - m`”；只能写“位置相位差依赖 `n - m`”。
- 不把 RoPE 圆盘说成 token 在句子平面移动；它是二维维度块的示意。
- 不把 gpt-oss、DeepSeek-V4 的公开证据泛化到所有模型。
- 删除 ChatGPT 独立口播字幕段；若后续提到专有模型，只能在审核说明中作为边界提醒，不能作为 RoPE 实现证据。
- 画面中不得出现制作侧提示词，例如“视觉焦点”“视觉爆点”“教学边界”“给普通观众的一句话”。
