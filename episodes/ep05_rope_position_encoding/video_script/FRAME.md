# EP05 RoPE 视频合成合同

## 版本目标

- 集数：EP05
- 主题：原始 Positional Encoding 与 RoPE 的位置建模差异
- 成片时长：目标约 216 秒，硬性下限 180 秒
- 核心体验：普通观众先看懂动画含义，再接受公式；公式只作为证据，不抢走理解入口
- 合成方式：HyperFrames 负责整体视频节奏、字幕和分镜；MATLAB R2021b 优先负责公式、Q/K 旋转和 RoPE 机制动画资产；最终审片音频必须使用授权参考音频生成的 `audio/voiceover.wav`

## 创意主张

原始 Positional Encoding 把位置加到输入里；RoPE 把位置转进 Q/K 的匹配关系，让 Attention 在比较时更直接看到相对位移 `n - m`。

## 证明物

| 用途 | 画面证明物 | 来源边界 |
| --- | --- | --- |
| 原始 PE | `x = x + pe` 与 sin/cos PE 公式 | 原始 Transformer 论文公式；Annotated Transformer 代码示例负责解释实现方式 |
| Attention 匹配 | `Attention(Q,K,V)=softmax(QK^T/sqrt(d_k))V` | 原始 Transformer 论文公式；Annotated Transformer 页面提供注释实现 |
| RoPE 机制 | `(\mathbf{R}_m q)^T(\mathbf{R}_n k)=q^T\mathbf{R}_{n-m}k` | RoFormer Eq. 12-16 语义 |
| 公开模型例子 | gpt-oss 可作为 RoPE 公开证据；DeepSeek-V4 可作为 Partial RoPE 公开证据 | 只引用公开资料，不推断专有模型内部实现 |
| 产品语境 | ChatGPT 只作为长上下文产品需求例子 | 只保留在审核说明里，不作为成片独立记忆点，也不写成 ChatGPT 使用 RoPE 的证据 |

## MATLAB 动画合同

- 默认版本：优先 MATLAB R2021b；若 R2021b 图形渲染超过 180 秒无有效输出、卡住、`-batch` 崩溃或触发 graphics/startup 异常，立即记录原因并切换到 R2026a 生成同一资产
- 参考边界：可以参考 `ep05_rope_position_encoding_225s_final_review.mp4` 里的 MATLAB 局部证明卡质感，但不得复用其中整页缩放、字幕遮挡或重叠布局
- 公式显示：必须使用 MATLAB 的 LaTeX/TeX 解释器或可验证的数学排版资产，不允许用字符拼接伪公式
- 防闪烁：公式、代码、正文说明优先作为稳定 PNG/HTML 静态层进入 HyperFrames；MATLAB 或 HyperFrames 动画只移动 Q/K 向量、箭头、局部高亮、相对位移条和计算填充条，不让文字、公式或整卡逐帧缩放/重排
- 圆盘与箭头：箭头必须从圆心出发并有明确端点；不能出现箭头与圆盘脱离造成的歧义
- 符号要求：使用 `θ_i`、`δ_i`、`mθ_i`、`nθ_i`，不要只用英文 `theta` / `delta` 替代；必要时在字幕或旁注解释读法
- MP4 稳定性：固定画布尺寸、固定坐标轴范围、固定文本框位置；禁止整页内容上下漂移、公式框抖动、字幕框抖动
- 动画重点：Q/K 旋转、相对位移 `n - m` 高亮、二维维度块分组、Annotated Transformer 输入加法示例
- 合成尺寸：MATLAB 资产进入 HyperFrames 时必须按当前视觉区重新构图；禁止把整页 MATLAB MP4 当作小图套进大框。若原始资产比例不匹配，应重渲对应比例或拆成公式、箭头、旋转块等局部资产。
- 动画边界：不得让公式、正文、字幕槽或整张卡片缩放、漂移、呼吸式脉冲；只允许箭头、Q/K 向量、相对位移连线、局部高亮和计数条运动。
- 公式资产：PE、Attention、RoPE 相对位移、2×2 旋转矩阵必须作为 MATLAB 渲染的局部数学资产插入对应场景；禁止用 HTML 字符拼出关键公式替代公式图。
- 视觉填充：每个场景主视觉区必须有公式、代码、计算结果、旋转图或类比图之一；不得出现空白页或大面积空白等待字幕补解释。
- 色彩要求：EP05 成片不使用绿色作为 RoPE 主色；统一使用深蓝、琥珀、酒红和暖纸色，字体与动画文本保持同一中文字体体系。
- RoPE 圆盘箭头要求：Q/K 箭头必须从圆心锚定出发，箭头头部不得超过向量长度的 18%，不得用大面积实心三角替代向量；Q 与 K 必须分色，箭头只表达二维维度块里的向量方向和旋转，不表达 token 在句子平面移动。
